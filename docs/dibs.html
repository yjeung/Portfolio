This is UI design concept of mobile application “Dibs”. This app isn’t launched, it was a project for a contest.

“Dibs” is a mobile application for a new way of trading used items.

In “Dibs,” users can buy, sell or donate items in groups that he or she follow.

Dibs

Description

1. logo and color theme

We picked Dibs! as the application's name, since the word is used to claim possession especially while within close members. It is used in Korean, too. We wanted the logo and its colors to look joyful but at the same time not to stand out too much, because we will use the color paletts for the interface as well as icons, for the sake of consistency.

Reflections

For this project, I and my team studied, researched and discussed a lot to learn how to create a good User Interface or how to develop an application. Still, looking back after sometimes, I realize that the process was not very efficient, and I also took part for it. I learned that to make an application complete, the first thing to do is to make it simple and that it's more efficient to learn a more appropriate tool than trying to stick with the program I know.
