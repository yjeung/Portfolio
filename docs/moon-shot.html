This is graphic design concept of mobile game “Moon Shot,” that I am currently working on this with my friends.

This game is a strategy game, in which players decide what to do for each day (study, lecture or publish) to reach the goal of shooting rocket before their rivals do.

Game UI

 

Description

My role was assets and UI design. Our team’s game designer shared an SWF prototype for communication so I could understand the mechanic and mood we have to convey.

 

I made a few animated sprites and other components such as buttons and icons. I also changed the layout so that it matches the ratio of mobile phones and is easy to control.  And for fun, I made logos of rivals as parodies of existing groups.

Reflection

At first, this was to be a quick project that had to be finished within a week, but time took longer than we thought. I could realize the difficulty and importance of predicting and managing time for a project, in this case especially when working together as a group.
