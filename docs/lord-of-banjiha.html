This is graphic design concept of mobile game “Lord of Banjiha,” which I am currently working on with my friends.

Banjiha means Semi-basement apartment, and is a name of our team “Banjiha Games”

This game is a mobile clicker game, where players tap to create more cockroaches and use items to speed the creation up. By level up, players get random new design of cockroaches.
